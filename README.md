# Power BI University Dashboard

A comprehensive Power BI dashboard for university institutional research and analytics.

## Overview

This Power BI dashboard provides insights across multiple domains:
- **Enrollment Profile** - Student demographics, enrollment trends, GPA analysis
- **Student Information** - Retention rates, persistence, course success
- **Student Course History & Grades** - Grade distributions, DFW rates, pass rates
- **Faculty Course Load** - Faculty workload analysis, section management
- **Degree History** - Degree conferral trends, time to degree metrics
- **Sections Overview** - Section capacity, fill rates, course offerings
- **Fees & Revenue** - Financial analysis, tuition dependency, revenue breakdown

## Files Included

- `Final Power Bi File.pbix` - Main Power BI dashboard file
- `README.md` - This documentation

## Data Sources

This dashboard connects to institutional data tables including:
- wly_ipeds (Enrollment/IPEDS data)
- stud_term_sum_div (Term summary data)
- stud_crs_history (Course history)
- j1_degree_history (Degree conferral)
- j1_faculty_load_table (Faculty workload)
- Sections (Course sections)
- Fees (Financial data)

## Key Metrics

### Enrollment
- Total Students
- Full-Time vs Part-Time Distribution
- Gender Balance
- Residential vs Commuter
- Average Career GPA

### Academic Performance
- Fall-to-Fall Retention Rate
- Term-to-Term Persistence
- Grade Distributions (A-F)
- DFW Rate
- Credit Success Rate

### Financial
- Total Revenue
- Tuition Revenue Dependency
- Average Fee Per Student
- Revenue Mix Analysis

### Faculty & Sections
- Faculty Load Distribution
- Section Fill Rate
- Capacity Utilization
- Courses by Department

## Requirements

- Power BI Desktop (Latest version recommended)
- Access to source data connections (may require VPN or network access)

## Usage

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Refresh data connections as needed
4. Use slicers (yr_cde, trm_cde) to filter by academic year and term
